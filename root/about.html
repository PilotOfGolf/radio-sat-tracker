Hello!

This is a webpage that you can use to get information about satelites, with an emphasiss on the Amateur Radio (ham) aspect. This website can get data that is helpful for contesting, making general QSOs, or even just to learn more about satelites.

This site uses the API from n2yo.com. I like this site, but I am not a fan of the UI. I understand why it is kept simple, becuase simple is easy to use. Because of this shortcoming, I plan to make a site that looks better, and provides the same data (because I am using their API).
